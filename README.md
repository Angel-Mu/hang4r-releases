# hang4r — downloads

**hang4r is a free macOS agents window that runs your Claude Code, Codex, and
Cursor CLI agents in parallel** — your own subscriptions, isolated git
worktrees, real diff review. No API keys, no markup, nothing to pay.

**[⬇ Download the latest release](https://github.com/Angel-Mu/hang4r-releases/releases/latest)** · [Website](https://hang4r.dev) · [Blog](https://hang4r.dev/blog/)

This repository hosts the signed, notarized builds and release notes.
The source code is private for now.

## Install

1. Download the `.dmg` from the [latest release](https://github.com/Angel-Mu/hang4r-releases/releases/latest).
2. Open it and drag **hang4r** to Applications.
3. Launch. Builds are Developer-ID signed and notarized by Apple — no
   Gatekeeper hoops.

**Requirements:** macOS (Apple Silicon), plus the agent CLIs you want to use,
installed and logged in — `claude` (Claude Code), `codex`, and/or
`cursor-agent`. hang4r spawns *your* CLIs; it never asks for API keys and
never touches your credentials.

**Updates:** hang4r checks for updates from this repository — you'll get new
releases in-app (Settings → Updates), always as a manual choice.

## Something broken? Ideas?

Open an [issue](https://github.com/Angel-Mu/hang4r-releases/issues) — bug
reports and feature requests both welcome. Please include your macOS version
and which agent CLI(s) you were using.
